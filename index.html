Tours and Travels Management

Type of Web Application website for booking different Travel Packages and Hotels

Tours and Travels Management is a web based travel and hotel reservation system developed using JAVA (JSP). In this system customer can choose an appropriate package of his choice. Once the package is chosen he can view certain options for the package with prices. As the customer is satisfied with any of the option he can submit the request to get a quote for the same. This request is received by the agent and he/she will review the same and publish the quote to the customer with various valuable offers. All the data regarding packages and prices is stored and accessed using MySQL. Also, the details of user, selected packages by user and quotes are getting stored in MySQL. Front-End is developed using HTML, CSS, JavaScript. For development and testing purpose this application was hosted on Apache Tomcat. For future purpose we did had a plan of adding Recommendation and reviews for more interaction of customer.

  • Technologies: JSP, Servlets, Java, JavaScript, HTML, CSS, MySQL,Apache Tomcat
